ü§´ Never Have I Ever - Instructions

üìã How to Get Started

  • Enter Username: Type your username (at least 2 characters)
  • Create Lobby: Click "Create Lobby" to start a new game room
  • Join Lobby: Enter a 6-character lobby code to join an existing game
  • Join Anytime: Players can join at any point during the game!

üéØ Game Flow

  • 15 Rounds: The game consists of 15 rounds
  • Topic Reveal: Each round presents a "Never have I ever" statement
  • Initial Voting (30s): Everyone votes Yes/No (have done it or not)
  • Player Selection: Game randomly picks one player
  • Confession (1 min): Selected player confesses if they were honest
  • Guessing (30s): Other players guess if confession is true
  • Results: See who guessed correctly and earn points

üèÜ Scoring System

  • Initial Vote: +1 point if you vote with the majority
  • Speaker Bonus: +3 points for each player who believes your confession
  • Correct Guessing: +3 points if you correctly guess the speaker's honesty
  • Final Winner: Player with the most points after 15 rounds wins!

üí° Tips

  • Sometimes it's strategic to lie in your initial vote!
  • Pay attention to other players' reactions
  • The confession phase is worth more points than initial voting
  • New players can join mid-game and start earning points
